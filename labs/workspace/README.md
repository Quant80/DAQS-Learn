# DAQS Labs — Interactive Sandbox

Welcome to your lab environment! You have Python 3.12, Node.js, and a full
data-science stack ready to go.

## Available Labs

| File | Topic | Difficulty | Time |
|---|---|---|---|
| lab_01_python.py | Python Fundamentals | Beginner | 30 min |
| lab_02_pandas.py | Data Analysis with Pandas | Intermediate | 45 min |
| lab_03_scraping.py | Web Scraping | Intermediate | 40 min |
| lab_04_sorting.py | Sorting Algorithms | Intermediate | 35 min |
| lab_05_fastapi.py | REST API with FastAPI | Advanced | 60 min |
| lab_06_ml.py | Machine Learning Intro | Advanced | 50 min |

## Quick Start

```bash
# Run a lab
python3 lab_01_python.py

# Interactive Python shell
ipython

# List all files
ls -lah
```

- **Data**: numpy, pandas, matplotlib, seaborn, scikit-learn
- **Web**: requests, httpx, fastapi, uvicorn
- **Tools**: git, curl, vim, nano, ipython
