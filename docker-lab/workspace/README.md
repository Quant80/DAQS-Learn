# DAQS Docker Lab

A real, isolated Docker environment — build images, run containers, and
use docker-compose without touching your host machine. Everything here
runs inside a nested, throwaway Docker daemon.

| Lab | Topic | Difficulty | Duration |
|---|---|---|---|
| `lab_01_dockerfile` | Write your first Dockerfile | Beginner | 20 min |
| `lab_02_images_containers` | Manage images & containers | Beginner | 20 min |
| `lab_03_volumes` | Persist data with volumes | Intermediate | 25 min |
| `lab_04_compose` | Multi-container apps | Intermediate | 30 min |

Each lab folder has its own `INSTRUCTIONS.md` — `cd` into it and `cat INSTRUCTIONS.md` to start.
